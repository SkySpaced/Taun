# Installation

---

### 📥 Installing Taunahi
* **Fabric Version:** `26.1.2 or 1.21.11`
* *Note:* If you already know how to install Forge/Fabric mods, you can skip this section and continue below.

### 🔌 Installing Fabric
1. Make sure you have previously run **Fabric 26.1.2 or 1.21.11** at least once.

### 📂 Installing Mods
1. Navigate to your local `.minecraft/mods/` folder.
2. Place the downloaded **Taunahi .jar** file directly into this folder.

### 🎮 Basic Usage
* Open settings in-game using the commands: `/taunahi` or `/ezss`
* The default keybind to open the **Start Scripts** menu is `B` (can be rebound in settings).

---

### ⚙️ Allocating More RAM (Optional)

This step is optional but can improve performance. By default, Minecraft uses up to 2GB of RAM, which may be insufficient.

**Steps to allocate more:**
1. Open the **Minecraft Launcher**.
2. Go to the **Installations** tab.
3. Hover over your profile, click the **three dots**, and choose **Edit**.
4. Click **More Options** at the bottom.
5. Find the **JVM Arguments** section. The first argument controls RAM, and looks like this:
   ```text
   -Xmx2G
   ```
6. If you want to allocate 6GB instead, change it to:
   ```text
   -Xmx6G
   ```

> ⚠️ **Warning:** Do not allocate excessive amounts of RAM, as this may reduce available memory for other apps.

---

### ⚡ Fix Slow Launch Times (Recommended)
Some bundled Java versions cause slow startup times. It's recommended to use the **Zulu JDK** instead of the default Java that comes with your launcher.

This can reduce launch times from 2–4 minutes to **15–30 seconds**, and improve performance in general.
