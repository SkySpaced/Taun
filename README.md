# Installation

---

### Installing Taunahi
Fabric 1.21.11. If you don't need to be shown how to install a Forge mod, you can leave this guide and continue.

### Installing Fabric
Make sure you've run Fabric 1.21.11.

### Installing Mods
After installing Fabric, go to your .minecraft/mods/. Place your Taunahi .jar file in this folder.

### Basic Usage
You can open settings using the commands /taunahi or /ezss. The default keybind to open the Start Scripts menu is B, but you can change this in the settings.

*   **Allocating More RAM (Optional):** Open the Minecraft Launcher, go to the Installations tab, edit your profile, click More Options, and change `-Xmx2G` to `-Xmx6G` (or more) in the JVM Arguments.
*   **Fix Slow Launch Times (Recommended):** Use the Zulu JDK instead of the default Java for faster startup (15–30 seconds vs. 2–4 minutes).

### Farming Scripts
Includes automatic, highly customizable scripts for:
*   **Wart & Crops:** High-uptime automation.
*   **Pumpkin & Melon:** Good for XP and money.
*   **Cactus & Cocoa Bean:** High-profit options.
*   **Mushroom:** Best for XP.
*   **Sugarcane:** Balanced option.

### Combat & Slayer Scripts
*   **Ghost:** High profit (up to ~45 mil/h).
*   **Crypt / Rev Slayer:** Fully AFK, pathfinding, auto-walk, and quest completion.
*   **Zealot (Ranged/Melee):** Efficient, smooth killing in the Dragon's Nest.
*   **Graveyard:** Ideal for new accounts.

### Foraging & Fishing
*   **Park/Hub/Island Foraging:** Automated tree cutting with failsafes and auto-sell.
*   **Auto-Fishing /w Travel:** Fully AFK fishing (water/lava/worms) with configurable paths.

### Miscellaneous & QOL
*   **Plot Cleaner:** Fully AFK plot clearing (walking/flying).
*   **Visitors:** Profit optimization for in-between tasks.
*   **Cobblestone/Alchemy/Harp:** Fast automation for quick gains.
*   **Anti-AFK:** Prevents being kicked.
*   **Fairy Souls & Cocoa Placer:** Utility features.

### Key Features
*   **Hydra Anti-Staff:** Advanced system for detecting and responding to staff.
*   **Scripting Sessions:** Detailed dashboard for statistics, event tracking, and earnings.
*   **Discord Integration:** Notifications and control via Discord.
*   **Remote Control:** Control scripts via dashboard or Discord.
*   **Humanized Movement & Menu:** Realistic movement and customizable HUD.
*   **Auto-Sell & Anti-Stuck:** Efficient inventory management and pathfinding.
*   **Failsafes & Rewarper:** Robust disconnection handling and automatic farm return.
*   **Proxy & Settings:** Built-in proxy support with hundreds of configuration options.
